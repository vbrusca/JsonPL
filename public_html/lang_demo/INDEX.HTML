Language Demonstration:

Note:
Earlier program demonstration/tests can be found in the "cfg" folder, ../cfg, directory. These should all work find but are deprecated with regard to the demonstration/testing of the language. The very earliest set of language tests can be found in the "EXEC4.HTML" file, ../EXEC4.HTML. The code ran in this file should work but is deprecated with regard to language demonstration/testing.

  • PRG001_ArgExample1
    This program demonstrates using a remote variable.
  • PRG002_ArgExample2
    This program demonstrates the {class} object's 'call' attribute using all four basic data types as arguments to the default function, 'main'.
  • PRG003_ArgExample3
    This program demonstrates simple use of arguments with the {class} object's default function, 'main'.
  • PRG004_ArgExample4
    This program demonstrates simple use of arguments that use the array data types, int[], float[], bool[], and string[].
  • PRG005_ArgExample5
    This program demonstrates using arguments of type array {ref} when passing parameters to functions.
  • PRG006_CallExample1
    This program demonstrates using the {call} object to call a function, main2, from another function, main.
  • PRG007_CallExample2
    This program demonstrates using three different techniques for calling the 'main2' function.
  • PRG008_CallExample3
    This program demonstrates calling the default function, 'call' attribute of the {class} object, of a library class.
  • PRG009_CallExample4
    This program demonstrates calling a specific function, specified in the short reference path, of a library class.
  • PRG010_CallExample5
    This program demonstrates using a remote function call.
  • PRG011_CallExample6
    This program demonstrates using three different techniques for calling the 'main2' function using a full reference string.
  • PRG012_CallExample7
    This program demonstrates calling the default function, 'call' attribute of the {class} object, of a library class using a full reference string.
  • PRG013_CallExample8
    This program demonstrates calling a specific function, specified in the full reference path, of a library class.
  • PRG014_DeclarationExampleAdv1
    This program demonstrates the declaration of arrays using the basic, strict, form.
  • PRG015_DeclarationExampleAdv2
    This program demonstrates the declaration of arrays using the advanced, non-strict, form.
  • PRG016_DeclarationExampleBasic
    This program demonstrates the declaration basic data type variables.
  • PRG017_RefExample1
    An example program that demonstrates using dynamic referencing to complete the reference string for some {ref} objects.
  • PRG018_RefExample2
    This program demonstrates using dynamic referencing.
  • PRG019_RefExample3
    This demonstration program shows a reference example that gets a value for a remote variable.
  • PRG020_RefExample4
    This program demonstrates using dynamic referencing with array variables and array elements.
  • PRG021_AsgnExample1
    This program demonstrates assigning a value to a local function variables. The variable is referenced by the {ref} object in the 'left' field of the {asgn} object.
  • PRG022_AsgnExample2
    This program demonstrates assignment using a {ref} object with a reference string that points to a class variable.
  • PRG023_AsgnExample3
    This example program demonstrates assigning a value to a class variable, tmpInt1, from a {ref} object that points to a function variables, '$.vars.rmpInt2', followed by an example with a remote, URL based, variable.
  • PRG024_AsgnExample4
    This example program demonstrates four different {asgn} object implementations using arrays and reference strings.
